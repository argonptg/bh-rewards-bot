# bh-rewards-bot

farms twitch rewards from brawlhalla dev streams

## Usage

Config:

```text
{
    "os": "linux/windows",
    "mode": "farm/harvest/register/login",
    "debug": true/false,
    "headless": true/false
}
```

`.env`:

```text
CLIENT_ID=<your client id>
CLIENT_SECRET=<your client secret>
```
